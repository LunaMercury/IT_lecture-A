〇 네트워크 기초 > OSI > 네트워크 기본

1. Network 는 무엇이며 무엇을 목적으로 하는가?
데이터 교환을 위한 기기간의 상호통신연결

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
LAN/WAN

3. LAN 의 특징에 대해서 서술하세요
비교적 가까운 거리의 기기들을 연결. 사내네트워크 등. 상대적으로 속도가 빠름. 유지보수가 간단. 독립적이라 보안성 좋음.  중앙집중식 관리가 가능

4. WAN의 특징에 대해서 서술하세요
인터넷 연결. LAN간의 연결. 대형인터넷공급업체 등이 제공. 광범위한 커버리지

5. InterNet 이란
전세계적으로 연결된 컴퓨터 네트워크. 여러 종류의 통신망을 한데 묶은 거대 통신망.

6. 프로토콜은 무엇인가?
네트워크 간 통신시 지켜야할 규정 및 규칙. 구문, 의미, 타이밍 등이 요소이다.

7. 네트워킹 모델의 목적은 무엇인가?
국가 및 기관 간 서로 다른 프로토콜 및 통신방법임에도 데이터 교환을 가능하게 하기 위한 방법. 네트워크를 구축하는데 필요한 설계도.

8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
8-1 : OSI 7
 (1) Psysical Layer : 전원 등이 들어와 있는지 등의 물리적인 상태 체크.
 (2) Data Link Layer : 오류 제어. 보낸 파일과 받는 파일 간의 동일성 체크. 장치간 데이터 운송방식 결정
 (3) Network Layer : 데이터 이동경로. 데이터가 이동하는데 여러 경로가 있을 수 있는데 그 것을 최적화.
 (4) Transport Layer : 오류제어. 받은 파일에 부족한 부분이 있는지 등을 확인 및 재요청 - 데이터 신뢰성.

8-2 : TCP/IP
 OSI 7에서 변화된 네트워크 모델
 (1) Psysical Layer
 (2) Data Link Layer
 (3) Internet Layer
 (4) Transport Layer


〇 네트워크 기초 > OSI > 1 Layer 물리계층

1. 물리계층 하는일/역할/관련 장치는?
 (1) 장치의 신호 등을 전달하고 받기 위해 전기적, 물리적 연결
 (2) 디지털 신호 전달
 (3)-1 LAN 케이블 : 주로 구리선. 네트워크 데이터 전송 담당
    -2 리피터 : 신호 증폭
    -3 허브 : 네트워크 장치 등을 연결

2. Cable 이란
(3)-1

3. LAN Cable 종류
이더넷 케이블 : UTP, STP 등이 있다.
광섬유 케이블 : 긴 거리 전송에 적합
동축 케이블 : TV선 같은 케이블

STP, UTP, FTP 등

4. 부호(Code) 란?
의미를 가지는 약속된 기호

5. 신호(Signal)이란?
역할 : 부호를 전달하기 위한 수단
약속 : 부호화(Encoding)
장치 : 케이블 등의 통신기기

6. 신호전달에 필요한 요소?
전기에너지, 통신망, 통신기기

7. 리피터란?
원거리 통신시 신호가 약해지는 문제를 해결하기 위해 신호를 증폭하는 기기

8. 허브란
여러 네트워크 장치 등을 연결


〇 네트워크 기초 > OSI > 2 Layer 데이터링크계층

1. OSI 모델중 2계층 모델이 하는 역할에 대해서 설명하세요
2계층 모델은 데이터링크 계층이다. 포인트 투 포인트간 신뢰성 있는 전송을 보장하기 위함. 즉 송신한 데이터와 수신한 데이터 사이 동일한 데이터인지 확인하기 위함이다.
▶ 역할 : 장치간(point to point) 데이터 운송방식 결정
▶ 프로토콜 : LAN, WAN

2. 데이터 링크의 목적달성을 위한 필요 요소 3가지는 무엇인가요
연결된 장비간 식별주소. ip주소 및 물리주소 등
오류 제어. 특정한 체크 데이터를 넣어 송수신 데이터 일치여부 확인.
흐름 제어. 속도제어 및 데이터 전송 전 미리 알림.

3. LAN/WAN 의 대표 2계층 프로토콜의 명칭은 무엇인가요
이더넷/HDLC

4. 초기 이더넷 프로토콜의 문제점은 무엇인가요
▶ 데이터 통신량이 많으면 충돌 영역이 발생. 일시적으로 통신이 불가능하게 됨.

5. 초기 이더넷 프로토콜이 사용하는 통신방법은 어떤것인가요
CSMA/CD

6. 초기 이더넷 프로토콜의 문제점을 해결하기 위한 장치는
L2 스위치


〇 네트워크 기초 > OSI > 3 Layer 네트워크기본

1. OSI 모델중 3계층 모델이 하는 역할은 무엇인가요
데이터 이동을 위한 최적 경로 탐색

2. 3계층 관련 장치는 무엇인가요
라우터, L3 스위치

3. 3계층 관련 프로토콜 종류는 어떤게 있나요
IP : 인터넷 기본 주소체계
ICMP : 목적지와 통신 가능 여부 확인
ARP : 통신을 위해 다음 기기의 주소 학습
Routing Protocol : 위를 바탕으로 최적 경로 선택

4. ICMP 의 용도는?
목적지와 통신 가능 여부 확인

5. IP 의 용도는?
▶ 논리적 주소를 통한 장치 식별

6. Routing 프로토콜의 용도는
최적경로탐색


〇 네트워크 기초 > OSI > 4 전송계층

1. OSI 모델중 4계층 모델이 하는 역할에 대해서 서술하세요 
`종단간`(end to end) 신뢰성 있는 데이터 전송을 담당.

2. 4 계층 관련 프로토콜은 어떤게 있고 각각 특징을 서술하세요
TCP 프로토콜 : 연결지향형 프로토콜. 신뢰성. 연결지향. 상대적속도느림.
UDP 프로토콜 : 비연결지향형 프로토콜. 빠른 요청및 응답. 여러 다수 지점 전송. 상대적속도빠름.
